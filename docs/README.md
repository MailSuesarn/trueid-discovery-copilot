# docs/

Place the project **hero image** here as `hero.png` — it is referenced at the top of the
root [README.md](../README.md).

Recommended specs for a GitHub README banner:
- **Aspect ratio:** ~3:1 (wide banner), e.g. **1536 × 512** or **1280 × 440** px
- **Format:** PNG (or JPG); keep under ~1 MB
- **Safe:** no real TrueID logo/trademarks — use a brand-neutral, original illustration

The image-generation prompt used to create it is kept in the conversation / commit notes.
